## NLP Desktop App
A Python desktop app built with Tkinter featuring user login and three NLP features 
powered by NLP Cloud API.

## Features
- User Registration & Login (JSON-based)
- Named Entity Recognition (NER)
- Sentiment Analysis
- Title/Headline Generation

## Setup
1. Install dependencies: pip install nlpcloud
2. Add your NLP Cloud API key in NLP.py
3. Create an empty db.json file: {}
4. Run: python app.py

## Tech Stack
Python, Tkinter, NLP Cloud API
